Chassis  0.0.1
Chassisはロボコンでの足回り制御を行うためのC++ライブラリである。
Omniクラスのリファレンス

N輪オムニの制御を行うクラス。

#include <Omni.h>

{C++}
template<int N>
struct Omni {
static_assert(N > 0);
template<class F> Omni(F&& f);
static constexpr int size() noexcept;
void calc(const Velocity& vel, float offset_rad = 0.0);
};

基本的な使い方

インスタンスの生成

  1. templateパラメータ N にモータ数を与える。
  1. モータへの出力を行うcallback関数を作成。
    callback関数の型はvoid(const float(&)[N])
    N要素float配列のconst参照を受け取り、voidを返す関数型。
  1. 引数にcallback関数を渡す。

Arduinoでの例

{C++}
Motor {
Motor(int, int);
void set_pwm(float);
};
Motor motor[3] {3, 4, 5, 6, 7, 8};
Omni<3> omni{[](const float(&pwms)[3]){
for (int i = 0; i < 3; ++i) {
motor[i].set_pwm(pwms[i]);
}
}};

Mbedでの例

{C++}
Motor {
Motor(PinName, PinName);
void set_pwm(float);
};
Motor motor[4] {D4, D5, D6, D7, D8, D9, D10, D11};
Omni<4> omni{[](const float(&pwms)[4]){
for(int i = 0; i < 4; ++i) {
motor[i].set_pwm(pwms[i]);
}
}};

動作

  1. 速度を表す構造体Velocityのインスタンスを生成。
  1. Velocityのインスタンスに速度をセット。
  1. move関数に渡す。
    {C++}
    Velocity vel{255, -255, 100};
    omni.move(vel);

詳細な説明

メンバ関数

Omni

コンストラクタ。 callback関数をセットする。

size

モータ数 N を返す。

calc

モータへのPWM出力を計算する。 その後callback関数にPWM出力を渡す。
